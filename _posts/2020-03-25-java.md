### 주석

![헬로월드 주석](/assets/헬로월드%20주석.png)

주석(comment): 프로그램에 대한 설명

#### 주석의 종류 3가지

첫번째.

/* ABC * /

/* 에서 * / 까지 무시

두번째.

/ \** ABC * /

/ \** 에서 * / 까지의 무시는 같으나 이것은 선언문 앞에서만 사용 가능

세번째.

// ABC

//줄의 끝까지 무시

### 클래스(class)

자바 프로그램은 클래스들로 구성.

![클래스 정의](/assets/클래스%20정의.png)
